1. Introdução

Um banco de dados MySQL é um sistema de gerenciamento de dados que utiliza a linguagem SQL (Linguagem de Consulta Estruturada) para armazenar, gerenciar e recuperar dados. O dados ficam armazenados em um servidor (pode ser local, na sua própria máquina) ou em um servidor central. Através de comandos inerentes à linguagem SQL, esses dados podem ser armazenados, tratados, explorados e analisados de forma rápida, ou exportados para serem tratados em Excel, por exemplo.

Nessa seção do Química Programada, vamos mostrar como formulários HTML e um Banco de dados em MySQL podem ser usados para elaborar um plano de manutenção preventiva de instrumentos e equipamentos.

Os dados que são inseridos na planilha do controle do plano de manutenção são salvos no banco de dados "manutenção" que foi criado no MariaBD, um banco MySQL criado no ambiente de desenvolvimento XAMPP. Fou criado um baco de dados chamado "manutenção" onde existe uma planilha chamada "map01" (manutenção preventiva 01) onde os dados inseridos são armazenados e podem ser acessados através de comandos da linguagem SQL.
