1. Introdução

O objetivo dessa página é compartilhar algumas aplicações de programação estruturada em HTML, CSS e PHP na área de Química e Engenharia Química. Esse material se destina a usuários já iniciados nessas linguagens, que saibam executar um programa php, usando um servidor local. Essa aplicação foi testada em Xampp v3.3.0.

Críticas e comentários podem ser feitos através do email quimicaprogramada@gmail.com.

A linguagem PHP integrada com formulários HTML é uma solução relativamente simples para geração de tabelas com dados de análises químicas que demandam cálculos simples. Esses relatórios podem ser impresos em PDF no próprio bowser de preferência do usuário ou pode ser instalada alguma biblioteca de geração de PDF em PHP como do DOMPDF ou outras disponíveis gratuitamente.

2. AnalyTech - Página inicial da calculadora química

No caminho criado pelo usuário no diretório C para instalação dos arquivos da aplicação Xampp, deverá ser criada uma pasta chamada AnalyTech (C:\xampp\htdocs\AnalyTech). A calculadora deverá ser salva nesta pasta. O programa salvo nessa pasta será acessado através do browser de sua preferência, após a instalação do servidor local, digitando na barra de acesso localhost/AnalyTech. Para aprender a usar um servidor local tipo Xampp, existe um vasto material na internet, recomenda-se os cursos online citados nas referências. Na página incial estão listados os diferentes módulos da calculadora. Cada módulo é apenas um exemplo de utilização de formulários HTML e PHP para cálculos na área de química que pode ser alterado para novas utilizações por necessidade do usuários usando um editor de texto gratuito como Sublime, ou mesmo o VSCODE, editando e salvando com a extensão .php. 3. AnalyTech - Determinação da massa específica e densidade de líquidos

A calculadora AnalyTech tem um módulo para cálculo da massa específica de uma substância líquida na temperatura de 20oC, apresentado na Figura 2. A massa específica é determinada pelo método do picnômetro. Nesta página pode-se ver que há uma entrada de dados para massa do picnômetro vazio, configurado para tratar dados com até 4 casas decimais, utilizando o "." (ponto) como separador. Também ha um formulário tipo select onde o usuário pode escolher entre 3 picnômetros A, B ou C, com volumes nominais hipotéticos de 50, 50 e 100 mL. Cada um tem o seu valor real de calibração que é possível ver no trecho de código da Figura 3. Nesse trecho podemos ver que foi feito um formulário HTML nos qual há como entrada massa do picnômtro (que é lida como variável "massavazio") e massa do picnômetro com produto (variável "massacomproduto"). Essas variáveis são recebidas quando o código é executado e enviadas pelo método "post" para um outro arquivo php chamado "Resultado da densidade.php" onde são recebidas e tratadas para cálculo da densidade. Na parte mais abaixo do código vemos que no mesmo formulário tem um trecho select onde há o nome do picnômetro escolhido que deverá aparecer no navegador "Picnômetro A - 50 mL", por exemplo, e a variável com a "tag" value que armazena o valor do volume de calibração do picnômetro que deverá ser alterado pelo usuário. Quando o picnômetro é escolhido, esse valor é armazenado na tag name="picnômetro[]", um array que é enviado ao arquivo php onde é calculada a massa específica. O resto do trecho do arquivo dessa página é um código HTML simples com parágrafos e tags h1, h2, e h3 com uma formatação de estilo especificada no arquivo "Formatação.css".

As variáveis de entrada do código da Figura 3 são enviadas pelo método post ao arquivo "Resultado da densidade.php" e são recebidas e tranformadas em novas variáveis que serão maipuladas para realização dos cálculos. A tag "massavazio" é enviada pelo método post e transformada na variável "$massavazio" que é usada para o cálculo da "$massaamostra",conforme mostrado na Figura 4 . O mesmo acontece com "massacomproduto". Ambos são transformados em "floats" através de do comando "floatval($massavazio)".

O envio e tratamento dos valores armazenados em "name=picnômetro[]" é feito através do comando "foreach" mostrado na Figura 5. O comando "if (isset($_POST["picnômetro"]))" indica que aquele trecho de código deverá ser executado apenas se tiver sido feita a seleão de algum picnômetro. Os dados são tratados, transformados em "floats" e manipulados para cálculo da densidade e formatados para serem exibidos com 4 casas decimais através do comando "$d2020format = sprintf('%.4f', $d2020)". Dentro desse loop foreach também há impressão das informações através do comando php "echo". As variáveis calculadas são impressas dentro das tabelas geradas por "echo table>tr>th>Massa específica (g/ml)/th>th>$densidadeformat/th/tr". Após o comando echo, as tabelas em php devem ser declaradas igualmente às tabelas dos códigos HTML. O resultado dos cálculos é exibido na Figura 6.

4. AnalyTech Envase gravimétrico de líquidos

O formulário de envase gravimétrico de líquidos usa como dados de entrada a identificação do líquido, a massa específica em g/ml e há um formulário de seleção das embalagens que foram produzidas e cujo peso dever ser estimado em kg. É apresentado na Figura 7. Com essas informações se obtém o peso de envase de cada tipo de embalagem e o peso de envase com uma tolerância de 0,5% como mostrado na Figura 8. No casos de embalagens em kg, também é possível calcular o peso com desvio de 0,5% selecionando a embalagem no formulário de seleção.

O código, nesse caso, se assemelha ao do formulário html de cálculo da massa específica, como pode-se ver na Figura 9, porém há dois "checkbox" com arrays distintos de coleta e envio de dados : um armazena os dados no array "embalagem[]" e outro no array "embalagemkg[]". Dessa forma, no arquivo que recebe esses dados pelo método "post", há dois "if (isset($_POST["embalagem"]))", cada um com um "foreach" para percorrer um array específico e executar operações com os dados, como mostrado na Figura 10.

5. AnalyTech Determinação da umidade de sais hidratados

A calculadora AnalyTech também tem um módulo de para cálculo da umidade de sais hidratados através da dissecação em estufa com uso de pesa-filtro. Foi esboçado um procedimento esperimental genérico que pode ser alterado no arquivo pelo usuário e um formulário para entrada de dados mostrado na Figura 11. Nesse formulário podemos entrar com a massa do peso-filtro (mr1 - massa do recipiente), a massa inicial do sólido mais o pesa-filtro (m1) e a massa final do sólido com o pesa-filtro após o final do tratamento na estufa(mf1). Devem também entrar com os dados mr2, m2 e mf2 e será obtido o valor médio.

Os dados de entrada são usados para o cálculo da umidade da amostra de sal que pode ser utilizada para comparar com a umidade do sal puro segundo previsto na estequiometria do mesmo para saber se essa amostra foi armazenada em condições adequadas e até mesmo ter uma ideia de seu grau de prureza. Esses resultados são apresentados na imagem da Figura 12.
